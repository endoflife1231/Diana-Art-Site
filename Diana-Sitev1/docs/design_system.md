# Design system — Diana final site

## 1. Концепт

**Главная идея:** сайт работает как «переводческая машина» для живописи Дианы: есть исходник — состояние, дневниковая фраза, городская сцена, фанатская привязанность; есть перевод — холст.  
**Визуальная метафора:** бумажный архив + камерная выставка + Telegram‑дневник, собранные в один редакционный интерфейс.  
**Эмоция:** личность, живость, немного дерзости, тепло, честность, ощущение настоящего художественного процесса.  
**Не должно напоминать:** шаблон Behance‑portfolio, стерильный white cube, «премиальный» сайт художника с пустыми словами, магазин постеров.

## 2. Направление дизайна

Итоговое направление: **editorial private gallery + zine archive + handmade digital catalog**.

- Editorial: крупная типографика, сильные заголовки, контрастные блоки.
- Private gallery: реальные работы в центре, крупные изображения, истории.
- Zine archive: живой дневниковый тон, неровная композиция hero, коллажность.
- Handmade: бумажные текстуры, рамки, слегка «неидеальные» повороты карточек.

## 3. Палитра

Палитра собрана из материалов: тёплая бумага, чёрная тушь, красный акцент, зелёный из «Родины Матери», приглушённый синий из «Залива».

## 4. Типографика

**Основной шрифт:** системный sans-serif stack для скорости и стабильности.  
**Акцентный:** Georgia/serif для крупных заголовков, чтобы сайт звучал как бумажный каталог, а не SaaS.  
**Запрещено:** много шрифтов, декоративный hand-written, капс в длинных текстах, мелкий серый текст на мобильных.  
**Заголовки:** большие, с плотной посадкой, editorial.  
**Основной текст:** ясный, без искусственной «поэтичной» туманности.

## 5. Сетка и композиция

- Контейнер: 1180px.
- Секции: `clamp(72px, 10vw, 140px)`.
- Карточки: большие радиусы 22–48px.
- Плотность: spacious, но каталог достаточно информационный.
- Hero: коллажная композиция, не шаблон «текст слева / картинка справа».
- Галерея: 12‑колоночная сетка с разной шириной карточек, чтобы избежать монотонности.

## 6. Компоненты

**Кнопки:** чёрные primary с красной тенью; ghost‑кнопки на бумажном фоне.  
**Карточки:** бумажный surface, тонкая граница, лёгкая тень, крупное изображение, статусный бейдж.  
**Фильтры:** pill buttons, active state чёрный.  
**Формы:** без backend, поэтому функция — подготовить сообщение для Telegram, а не имитировать отправку.  
**Модалка:** image + story, закрытие по кнопке, overlay и Escape.  
**Навигация:** sticky, mobile burger.

## 7. Motion

- Плавное появление секций.
- Мягкий hover карточек.
- Ротация фраз в hero как «переводчик состояний».
- Back‑to‑top появляется только после скролла.
- `prefers-reduced-motion` отключает продолжительные transition/animation.

## 8. Tone of voice

Писать как живой художественный дневник: конкретно, умно, с лёгкой иронией.  
Избегать «магия искусства», «погрузитесь», «уникальный опыт», «изысканная коллекция».  
Формальность: низкая/средняя.  
Поэтичность допустима, если она точная.  
Ирония допустима и нужна, потому что она есть в исходных материалах.

## 9. Референсы и источники решений

- Приложенный Telegram‑экспорт: интонация, дневниковость, живой авторский голос.
- Старая версия сайта/GitHub: структура каталога, метаданные работ, интерактивные идеи.
- Creative Bloq: портфолио должно показывать не только работы, но и личность/подход; сильные visual-first портфолио дают работам говорить первыми.
- Локальные изображения работ: цветовая система и композиционная энергия взяты из реальных холстов.
