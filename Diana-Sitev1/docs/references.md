# References — Diana final site

## Нравится

### Референс 1 — Telegram‑портфолио Дианы

**Источник:** `Диана/ChatExport_2026-06-11/messages.html` и связанные фотографии.  
**Что нравится:**

- композиция: посты как последовательный дневник;
- палитра: реальная палитра работ, без искусственного брендбука;
- типографика: живой текст, разговорные формулировки;
- карточки: каждая работа имеет историю;
- атмосфера: честно, смешно, иногда резко, не стерильно.

**Что НЕ копировать:** интерфейс Telegram как UI. Сайт должен быть сайтом, а не скриншотом канала.

### Референс 2 — первая версия сайта / GitHub

**Источник:** `diana_site/index.html`, `diana_site/script.js`, `diana_site/style.css` и репозиторий `endoflife1231/Diana`.  
**Что нравится:**

- уже собранный каталог;
- фильтры, поиск, модалки;
- структура работ с описаниями;
- направление «живопись как перевод состояния».

**Что НЕ копировать:** слишком спокойную, равномерную лендинговую композицию; заглушки; фейковый email; недостаточную уникальность hero.

### Референс 3 — editorial / work-first portfolios

**Источник:** Creative Bloq, материал про портфолио: https://www.creativebloq.com/portfolios/examples-712368  
**Что нравится:**

- портфолио должно показывать личность и подход, а не только список работ;
- визуально сильные портфолио часто дают работам говорить первыми;
- чистая структура помогает не потерять контакт и CTA.

**Что НЕ копировать:** generic designer portfolio layout без художественного голоса.

## Не нравится

### Антиреференс — шаблонный сайт художника

**Что не нравится:**

- стерильный white cube без личности;
- «премиальная» бежево‑золотая эстетика ради статуса;
- карточки‑клоны без истории;
- фейковый маркетинг и общие слова;
- слишком много декоративных эффектов;
- невнятные CTA.

## Итоговые правила

Делать:

- показывать реальные работы крупно;
- сохранять авторский дневниковый голос;
- делать сайт чуть неровным и живым, но технически аккуратным;
- отделять статус покупки от художественного текста;
- держать контакты быстрыми.

Не делать:

- имитировать галерею, которой здесь нет;
- скрывать работы за тяжелой анимацией;
- использовать случайные стоковые изображения;
- подменять Диану абстрактным «брендом художника».
