# Документация изменений: правки 3

## Запрос

Внести новый пакет правок по папке `правки 3`:

- удалить нижнюю текстовую кнопку `Наверх`, оставить только вертикальную;
- сделать фотографии в правой части архива открываемыми;
- уменьшить и минимализировать вертикальную кнопку `Наверх`;
- убрать стрелку из блока `#наличие`, оставить мягкое выделение;
- доработать блок `О Диане`: сделать вертикальную подпись заметнее, вернуть обычную жирность текста справа, оставить высокий контраст и красную вертикальную линию;
- вынести фразу `не слова / состояния` из наложения на фото/видео;
- сделать hero-видео визуально ближе к минималистичным карточкам рядом;
- мягко оптимизировать сайт от подлагиваний без критической перестройки.

## Изменения в коде

### `index.html`

- Удалена нижняя ссылка `Наверх` из footer.
- Изображения архивных карточек обёрнуты в кнопки с `data-open-image`, `data-image-title` и `data-image-alt`.

### `js/script.js`

- Добавлена функция `openImageModal()` для просмотра архивных изображений в существующем `<dialog>`.
- Добавлен обработчик клика по `[data-open-image]`.
- Кнопка `Наверх` теперь учитывает `prefers-reduced-motion`.
- Подсветка активного пункта навигации переведена на `requestAnimationFrame`, чтобы не пересчитываться напрямую на каждый scroll-event.

### `css/style.css`

- Вертикальная кнопка `Наверх` уменьшена и упрощена: меньше высота, тише тень, тоньше акцент.
- У `#наличие` убран круглый значок-стрелка; вместо него добавлена тонкая красная линия и мягкий нижний акцент.
- Текст блока `О Диане` возвращён к обычной жирности, но сохранён тёмным и читаемым.
- Вертикальная подпись `О Диане` сделана крупнее и контрастнее.
- Hero-видео оформлено как лёгкая карточка, без тяжёлой чёрной рамки.
- Плашка `не слова / состояния` перенесена в свободную область hero-коллажа.
- Для карточек и hero-элементов добавлен `contain: paint`, чтобы снизить лишнюю перерисовку.
- Добавлены стили для кнопок архивных изображений и состояния hover/focus.
- Добавлен отдельный режим модалки для архивных изображений.
