# Web Site Skill Pack

Набор markdown-файлов для повторяемого создания качественных сайтов через ChatGPT/Claude/агента.

## Файлы

- `web_site_skill.md` — ядро правил и рабочий протокол.
- `brief_template.md` — шаблон брифа для нового сайта.
- `design_system_template.md` — шаблон дизайн-системы и арт-дирекшна.
- `references_template.md` — шаблон для референсов и антиреференсов.
- `quality_checklist.md` — QA-чеклист перед финальной выдачей.
- `task_prompt_template.md` — короткий шаблон промпта для конкретной задачи.
- `agent_instructions.md` — когда подключать агента и как формулировать задачу.

## Как использовать

1. Прикрепи весь набор или минимум `web_site_skill.md` + `quality_checklist.md` к новому чату.
2. Добавь заполненный бриф или используй `task_prompt_template.md`.
3. Напиши: «Работай по `web_site_skill.md`. Создай сайт по брифу и прикреплённым файлам».
4. По возможности прикрепи изображения, тексты, референсы и антиреференсы.

## Рекомендуемый минимум для обычной задачи

- `web_site_skill.md`
- `brief_template.md` или заполненный бриф
- `quality_checklist.md`
- материалы проекта: архивы, изображения, тексты

## Рекомендуемый набор для сложной задачи

- все `.md` файлы из этого пакета;
- отдельный файл с контентом;
- отдельная папка/архив с изображениями;
- референсы;
- технические ограничения;
- желаемый формат результата.
