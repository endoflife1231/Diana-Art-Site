# Документация изменений от 2026-06-14

## Цель
Вернуть и повторить правки из Cursor-чата, исправить найденные ошибки, улучшить UX, доступность, визуальную выразительность и добавить видеоконтент без усложнения статического сайта.

## Бэкапы
- Старая версия перед правками: `backups/Diana-Sitev1-old-20260614T203855Z/`.
- Новая версия после правок: `backups/Diana-Sitev1-new-20260614T203855Z/`.

## Изменения в HTML
- Заменены черновые тексты в шапке, hero и footer на более нейтральные production-формулировки.
- Исправлена фраза «Красота всегда будет в глазах смотрящего».
- Переписан резкий абзац в блоке «О Диане» на текст, соответствующий художественному тону сайта.
- Исправлена сломанная разметка цитаты в архиве: вместо незакрытого `<p>` используется оформленный `blockquote` с `cite`.
- Добавлены две видео-карточки в архив: рутинный процесс и February edit.
- Подключен `js/data.js` перед основным скриптом для отделения данных каталога от логики.
- В форму добавлена кнопка копирования подготовленного сообщения.

## Изменения в JavaScript
- Каталог работ теперь читается из `window.__SITE_DATA__.artworks`, а не хранится только внутри основного скрипта.
- Карточки галереи открывают модальное окно по клику по всей карточке, а также по Enter/Space.
- Карточки в блоке «Наличие» также открывают модальное окно и доступны с клавиатуры.
- Добавлен focus trap внутри `<dialog>`, чтобы Tab не уводил фокус за пределы модального окна.
- В модальном окне для работ `february` и `zaliv` добавляется связанное видео.
- Форма контакта сохраняет подготовленное сообщение и позволяет скопировать его через Clipboard API с fallback-сообщением.
- Scroll-обработчик переведен на `requestAnimationFrame`, чтобы снизить нагрузку на FPS.

## Изменения в CSS
- Отключен тяжелый `backdrop-filter` в header и backdrop модального окна.
- Навигация получила более выразительное underline-состояние для hover/focus/active.
- Hero-коллаж исключен из reveal-анимации, чтобы убрать визуальное «дергание».
- Метка hero стала контрастнее: светлый фон, темный текст, красная рамка.
- Статистика получила красные акцентные цифры и легкое hover-состояние.
- Блок «О Диане» выровнен по центру колонок, на desktop kicker сделан вертикальным.
- Номера manifesto оформлены как красные круглые маркеры.
- Добавлены стили для видео-карточек архива и видео внутри модального окна.
- Добавлены focus-visible состояния для кликабельных карточек.

## Проверки
- `node --check Diana-Sitev1/js/data.js` — синтаксис данных валиден.
- `node --check Diana-Sitev1/js/script.js` — синтаксис основного JS валиден.
- `python3 -m http.server 4173` + `curl -sI http://127.0.0.1:4173/Diana-Sitev1/index.html` — локальная отдача страницы возвращает HTTP 200. Скриншот не сделан: в контейнере нет Chromium/Firefox/Playwright.
