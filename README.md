# Diana Art Site

Статический сайт-портфолио художницы Дианы: каталог работ, дневниковые истории, архивные материалы, блок наличия и контакты.

Сайт задуман не как стерильная галерея, а как небольшой художественный маршрут: от работы к истории, от истории к состоянию, которое можно узнать без лишнего объяснения.

## Live

GitHub Pages:

https://endoflife1231.github.io/Diana-Art-Site/Diana-Sitev1/

## О проекте

Главная идея сайта - живопись как перевод состояний. В интерфейсе это собрано через:

- коллажный hero-блок с работами, портретом и видео;
- каталог картин с фильтрами, поиском и сортировкой;
- модальные карточки с историей каждой работы;
- отдельный блок работ в наличии;
- архивный раздел с контекстом из дневниковой среды;
- контакты для Telegram и TikTok.

## Стек

Проект сделан без сборщика и фреймворков:

- HTML
- CSS
- JavaScript
- GitHub Pages

Сайт можно открыть локально напрямую через `index.html`.

## Структура

```text
Diana-Sitev1/
  index.html
  css/
    style.css
  js/
    data.js
    script.js
  assets/
    icons/
    images/
    videos/
  docs/
  robots.txt
  sitemap.xml
  .nojekyll
```

## Где менять контент

- Работы, статусы, цены, описания и категории: `Diana-Sitev1/js/data.js`
- Основные секции, SEO и внешние ссылки: `Diana-Sitev1/index.html`
- Визуальный стиль, адаптив и анимации: `Diana-Sitev1/css/style.css`
- Изображения и видео: `Diana-Sitev1/assets/`

## Возможности

- адаптивная вёрстка для desktop и mobile;
- фильтры по статусам и темам;
- поиск по работам;
- сортировка каталога;
- модальные окна для работ и архивных изображений;
- остановка видео при закрытии модального окна;
- кнопка включения звука в hero-видео;
- поддержка `prefers-reduced-motion`;
- Open Graph, Twitter Card, sitemap и robots для публикации.

## Деплой

Проект опубликован через GitHub Pages из ветки `main`, папка `/ (root)`.

Так как сайт лежит в подпапке `Diana-Sitev1`, публичный URL включает этот путь:

```text
https://endoflife1231.github.io/Diana-Art-Site/Diana-Sitev1/
```

Если переносить сайт в корень репозитория или подключать кастомный домен, нужно обновить URL в:

- `Diana-Sitev1/index.html`
- `Diana-Sitev1/robots.txt`
- `Diana-Sitev1/sitemap.xml`

## Примечание

Форма на сайте не отправляет данные на сервер. Она готовит текст сообщения, который можно использовать для связи с Дианой в Telegram.
